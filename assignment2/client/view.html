This is a lightweight app used for keeping track of a list of courses (name and description).

Once a new course is submitted, you will view it on the list below the form.
You can view its details (unique ID assigned from backend, given name, and currnet description).
You can change the description (but not the name) using the input field on the appropriate line, then clicking submit new desc.
You can also delete the course entirely which will remove it from all back end records and remove it from the list.

Unique ID's are auto assigned upon submission of form by the backend.
Created times and updated times (for desc updates) are tracked as well for reference.

NOTE that the order shown will not necessarily stay the same after a new update to the database (upon changing description or deleting course).
